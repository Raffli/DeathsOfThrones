Deaths of Thrones

Deaths of Thrones is a study project developed by Nikolai Stecker and Florian VÃ¶lkers.
It was developed for the module "Relationale Datenbanken" of the bachelor degree course Media Systems at HAW Hamburg.

The goal was to develop a web application on basis of a database using MySQL. The database started out with a data-set of the most important 100 deaths in first 6 seasons of the TV-Show Game of Thrones. The developers added data about the murderers, locations and episodes surrounding those deaths manually from a Game of Thrones Wiki.

For the development of this Web Application the following development tools were used:

  • Angular: Typescript-based Frontend-Development Framework
  • Chart.js: Library for Charts
  • GitHub: Filehosting with Version Control
  • MySQL: relational database management system
  • PrimeNG: UI library for Angular
  • Spring: Java Application Framework
  • Spring Tool Suite: IDE for Spring Development
  • Webstorm: IDE for Web Development
